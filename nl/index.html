BeatleTech is mijn (Coen Stevens) freelance eenmanszaak waar anderen gebruik kunnen maken van mijn diensten, vaardigheden en ervaringen in het ontwikkelen van web applicaties en oplossen van technische uitdagingen.

Software Ontwikkeling

Als ondernemer, onderzoeker en programmeur, die zojuist is teruggekeerd uit silicon valley om het nederlandse weer te verkiezen boven het zonnige San Francisco, ben ik nu op zoek naar interessante job opportunities en uitdagingen om jouw bedrijf te helpen, waar ik beschikbaar voor ben als freelancer.

Naast all-round web-development in Ruby, Node.js, Clojure en PHP, ben ik gespecialiseerd in:

  • Statistiek & Analyse
  • Schaalbare architecturen
  • Big data
  • Cognitieve wetenschap & Intelligente systemen
  • Recommender systems (aanbevelings systemen)
  • NoSQL databases
  • Functioneel programmeren
  • Business Intelligence & Data warehousing

Advies

Naast ontwikkelen kan ik ook advies geven over ontwikkeling en bedrijfs strategiën. Als oprichter van een Y Combinator bedrijf heb ik mijn aandeel wel gehad in de emotionele achtbaan waar iedere ondernemer doorheen gaat, wat ik graag wil delen en onderwijzen aan anderen.

Ervaring

Ik heb gewerkt aan een hoop verschillend applicatie, en met verschillende technologiën van welke ik de meest noemenswaardige hieronder vermeld heb.

Bij Wakoopa heb ik hun recommender system ontworpen, geoptimaliseerd en uitgebreid in Ruby, JRuby en Clojure, en daarnaast geontwikkeld in Ruby on Rails voor software classificatie, versie nummer herkenning en dashboard tools. Bij Newcope ben ik verder gegaan met het werken aan dashboard analytics met Ruby on Rails.

Naast Ruby on Rails heb ik een hoop ervaring opgedaan met Javascript (JQuery), en javascript aan de server kant genaamd Node.js. Voor Newcope heb ik volwaardig penny veiling system gebouwd, wat een ideale fit was voor de asynchrone architectuur van Node.js.

Clojure is waar ik de meeste tijd aan spendeer voor zowel hobby projecten als voor mijn vorige werk bij Global Orange aan data warehouse oplossingen en data import parsers.

In mijn eerdere dagen heb ik veel gewerkt met Java, waar een van mijn grotere projecten bestond uit het bouwen van een uren registratie systeem. Tegenwoordig gebruik ik veel java libraries in combinatie met Clojure, en gebruik veel van de Apache foundation projecten, zoals ActiveMQ, Solr, Maven, etc.

Bij mijn vorige baan als IT architect heb ik ook PHP applicaties ontworpen en veel aan troubleshooting gedaan. Hiervoor zijn we al van start gegaan met het Symfony 2.0 framework wat gepland staat voor een final production ready release in Maart 2011.
